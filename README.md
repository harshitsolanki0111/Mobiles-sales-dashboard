## ‚öôÔ∏è Getting Started

### Prerequisites

- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/motorola-sales-dashboard.git
   cd motorola-sales-dashboard
   ```

2. **Open the dashboard**
   - Launch Power BI Desktop
   - Open `Motorola_Sales_Dashboard.pbix`

3. **Refresh data** *(if using live data)*
   - Go to **Home ‚Üí Transform Data ‚Üí Refresh**

4. **Explore**
   - Use the left-panel **month slicers** and bottom **filter dropdowns** (Mobile Model, Payment Method, Brand) to interact with the data
